# Fonts Directory

Place your antique serif font files (TTF or OTF format) in this directory.

The default font path is configured in settings, but you can specify a custom font path in the settings menu.

## Recommended Fonts

- **EB Garamond** - Classic serif font
- **Crimson Text** - Elegant serif
- **Lora** - Readable serif
- **Playfair Display** - Decorative serif

Download fonts from Google Fonts or other font repositories and place them in this directory.
